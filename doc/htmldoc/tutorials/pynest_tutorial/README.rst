PyNEST tutorial
===============

The PyNEST tutorial series guides you through the first steps of using the NEST
Simulator, from creating neurons and connections to building spatially
structured networks. Each part is an executable script: you can read it here,
download it as a Python file, or run it as a Jupyter notebook.
