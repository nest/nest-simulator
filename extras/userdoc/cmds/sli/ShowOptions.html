Name:
ShowOptions - Display all options for a given name
Synopsis: 
/f  ShowOptions -> --
Diagnostics: 
None
Author:

Description:

  availability of the function, if the function is defined in some  
  "namespace" dictionary.  
  For example: "arr::Mean".
