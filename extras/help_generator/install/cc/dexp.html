Name:
dexp - computes an integer power of 2 and returns the result as double
Synopsis: 
integer dexp -> double
Examples: 
-5 dexp frexp -> 0.5 -4
Description: 
 
  This is an interface to the C++ expression  
  double std::ldexp(1.0,int)
